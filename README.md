# Uber Trip Analysis Dashboard 🚖📊

## Overview

An interactive Power BI dashboard developed to analyze Uber booking performance, revenue trends, customer ratings, and operational insights.

## Business Requirements

The project includes a Business Requirements Document (BRD) containing:

* Business Problem Statement
* Stakeholder Requirements
* Use Cases
* KPIs and Metrics
* Dashboard Objectives


## Key Metrics

* 93K Completed Bookings
* 52M Total Revenue
* 2.51M Distance Covered

## Features

* Booking Trend Analysis
* Vehicle-wise Revenue Analysis
* Customer & Driver Rating Analysis
* Location-wise Performance Tracking
* Interactive Filters and Navigation

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
